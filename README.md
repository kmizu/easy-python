# 楽しくPythonを学ぼう

## 概要

このプロジェクトは、プログラミング初心者がPythonを楽しく学べる入門書です。
コンピュータの基礎から始めて、最終的にはPythonの文法をBNF（Backus-Naur Form）レベルで理解できるまで、段階的に学習を進めていきます。

## 特徴

- **完全初心者対応**: コンピュータの基礎から丁寧に解説
- **実践的な例題**: 家計簿、連絡先帳、商品管理など身近なプログラムで学習
- **段階的にBNF記法を導入**: 最初は簡単な説明から始め、徐々に本格的な文法理解へ
- **豊富な実行例**: 【実行】タグ付きの実行可能なコード例を各章に配置
- **エラーを恐れない学習**: エラーメッセージの読み方も積極的に解説

## 使い方

### 環境構築

#### 推奨方法：Makefileを使用

```bash
# プロジェクトルートで実行
make setup
```

このコマンドで以下が自動的に実行されます：
- Python仮想環境の作成
- 必要な依存関係のインストール
- MkDocsとプラグインのセットアップ

#### 手動セットアップ

```bash
# Python仮想環境の作成
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 依存関係のインストール
pip install -r requirements.txt
```

### 開発サーバーの起動

```bash
# Makefileを使用（推奨）
make serve

# または手動で実行
venv/bin/mkdocs serve
```

ブラウザで http://127.0.0.1:8000/ にアクセス

### 静的ファイルの生成

```bash
# Makefileを使用
make build

# または手動で実行
venv/bin/mkdocs build
```

生成されたファイルは `site/` ディレクトリに保存されます。

### その他の便利なコマンド

```bash
# コード品質チェック
make validate     # 全チェック実行
make format      # Blackでコード整形
make lint        # flake8でリント
make typecheck   # mypyで型チェック

# 依存関係の確認
make deps

# クリーンアップ
make clean       # 生成ファイルを削除
make clean-all   # 仮想環境も含めて削除

# ヘルプ
make help        # 利用可能なコマンド一覧
```

## 対象読者

- プログラミング初心者でPythonを学びたい方
- Pythonは使えるが、より深く理解したい方
- 言語設計や文法に興味がある方
- BNF記法やコンパイラ理論に興味がある方

## 学習の進め方

1. **順番通りに読む**: 章は段階的に構成されているため、順番通りに進めることを推奨
2. **コードを実際に動かす**: 各章の実行例は必ず自分の環境で試してみる
3. **BNF記法に慣れる**: 最初は難しく感じるかもしれませんが、徐々に慣れていきます
4. **エラーを恐れない**: エラーメッセージも学習の一部として活用する
